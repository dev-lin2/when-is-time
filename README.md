# When Time Is?

Simple timezone utility app built with React + Vite + TypeScript.

## What It Does

- `When Time Is` widget:
  - Pick a time and timezone identifier (like `UTC`, `GMT`, `EST`)
  - Click `Calculate`
  - See how far that time is from your local time
- `World Clock Dashboard`:
  - Live clocks with second-by-second updates
  - Add/remove timezone cards
- `Time Comparison`:
  - Compare one date/time between two timezone identifiers
  - Live countdown display
- Light and dark theme support

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- TanStack Router
- date-fns / date-fns-tz

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Scripts

- `pnpm dev` - start dev server
- `pnpm build` - production build
- `pnpm preview` - preview build
- `pnpm lint` - lint source files
- `pnpm typecheck` - TypeScript check
